A practical workshop aimed at those new to writing client-side web applications and using the ClojureScript programming language.

ClojureScipt is a lisp style programming language which live compiles to JavaScript for an effective development workflow with instant feedback.

ClojureScipt can be used for simple landing pages with CSS frameworks, Single Page Apps following the react.js approach or complex user interfaces with effective state management.

This workshop guides you through ClojureScript, figwheel-main, reagent and the basics of re-frame, with common design approaches common covered along the way.
